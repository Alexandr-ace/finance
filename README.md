# 🕷️ Парсер Wildberries с сохранением в PostgreSQL

Простой и надежный парсер для сбора данных о товарах Wildberries. Проект полностью контейнеризирован с помощью Docker, что гарантирует его запуск на любой машине без дополнительной настройки окружения.

## 🚀 Возможности
- Сбор данных о товаре (артикул, название, цена)
- Надежное хранение данных в PostgreSQL
- Полная изоляция зависимостей через Docker
- Защита чувствительных данных через `.env`

## 🛠️ Стек технологий
- **Python 3.12+**
- **PostgreSQL 15** (база данных)
- **Docker & Docker Compose** (контейнеризация)
- **Библиотеки:** `requests`, `beautifulsoup4`, `psycopg2-binary`, `python-dotenv`
